# FluxShowcaseApp

See FluxUI in action — live, interactive, and production-ready.

Key Features:

Component gallery: Live preview of all Atoms, Molecules, Organisms

Theme switcher: Toggle Light/Dark + multi-app themes

MVVM examples: Payment flow, Login flow, Dashboard flow

Do’s / Don’ts guides: Demonstrates proper usage of tokens and components

Real app scenarios: Shows reusable, production-ready workflows
